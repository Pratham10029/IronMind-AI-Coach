# 💪 IronMind AI Coach

IronMind AI Coach is an AI-powered fitness and gym assistant built using Streamlit, Groq API, and SQLite.

The application helps users with:
- Personalized fitness guidance
- Workout recommendations
- Diet and nutrition support
- Supplement suggestions
- Workout tracking
- Weight progress monitoring
- Chat-based AI coaching
- Fitness streak tracking

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ironmind-ai-coach.git
```

### 2️⃣ Move Into Project Folder

```bash
cd ironmind-ai-coach
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Add Your Groq API Key

Create an environment variable:

```bash
GROQ_API_KEY=your_api_key_here
```

Or add inside your Streamlit secrets.

---

## ▶️ Run the Application

```bash
streamlit run gym/gym.py
```
